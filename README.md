# Reporte-Machine-Learning-servicio-social
Este es el reporte final de mi servicio social para obtener el título de técnico en computación. Dicho trabajo consistió en aprender las bases del desarrollo de modelos de Machine Learning, y para ello, se elaboró un modelo que buscaba detectar la posibilidad de que un paciente desarrollara una enfermedad coronaria en los próximos 10 años.
